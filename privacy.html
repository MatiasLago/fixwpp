Política de Privacidad — FixWpp

Vigente desde: 06/10/2025

Resumen

FixWpp es una extensión para Google Chrome que permite ocultar el panel lateral de WhatsApp Web y expandir la conversación a pantalla completa. Esta extensión se ejecuta exclusivamente dentro del navegador del usuario y no recopila, almacena ni transmite datos personales.

Datos que recopilamos

No recopilamos ningún tipo de información personal ni técnica del usuario (por ejemplo, nombres, direcciones de correo, IP, historial de navegación, contenido de mensajes, métricas de uso o identificadores).

Permisos del navegador

  • activeTab: Permite interactuar con la pestaña activa del usuario al accionar la extensión.
  • storage: Guarda de forma local la preferencia del usuario (mostrar/ocultar panel). No se envían datos fuera del dispositivo.
  • scripting y host_permissions en https://web.whatsapp.com/*: Inyecta el script que ajusta la interfaz de WhatsApp Web.

Estos permisos no se usan para extraer ni compartir información del usuario.

Cookies y servicios de terceros

FixWpp no instala cookies propias ni integra SDKs o scripts de terceros, y no comunica datos a servidores externos.

Menores de edad

La extensión no está dirigida específicamente a menores y no procesa información personal de ningún usuario.

Cambios a esta política

Podremos actualizar esta política para reflejar mejoras o requisitos legales. La fecha de vigencia se actualizará en la parte superior.
